# Игра «Змейка» (Snake Game)

Учебный проект на C++ с использованием библиотеки SFML.

## Описание
Классическая игра «Змейка», где игрок управляет змейкой, собирает еду и набирает очки. Реализовано сохранение рекорда.

## Как запустить
1. Установите библиотеку SFML.
2. Скомпилируйте проект с помощью вашего IDE (например, Visual Studio).
3. Запустите исполняемый файл.

## Технологии
- C++17
- SFML Graphics
